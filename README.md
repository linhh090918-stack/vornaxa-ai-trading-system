# Vornaxa AI Trading System

This is a deployable React + Vite website for Vornaxa AI Trading System.

## Local preview

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Upload this folder to a GitHub repository.
2. In Vercel, click Add New > Project.
3. Import the GitHub repository.
4. Framework Preset: Vite.
5. Build Command: npm run build.
6. Output Directory: dist.
7. Click Deploy.
8. Add domains: vornaxaai.com and www.vornaxaai.com.
